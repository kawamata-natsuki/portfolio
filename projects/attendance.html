勤怠管理アプリ Personal

期間

2025/04 〜 2025/05（1ヶ月半）

概要

プログラミングスクールCOACHTECHの課題として制作した勤怠管理アプリです。 ログイン・出退勤打刻・勤怠一覧・修正申請・管理者承認など、勤怠管理に必要な基本機能を実装しました。

機能一覧

一般ユーザー機能

  • 会員登録（Fortify + メール認証）
  • ログイン / ログアウト
  • バリデーション（FormRequest）、エラーメッセージ表示
  • 出勤 / 退勤 / 休憩打刻（1日1回制御、複数休憩対応）
  • 勤怠ステータス表示（勤務外 / 出勤中 / 休憩中 / 退勤済）
  • 勤怠一覧表示（月切り替え、詳細画面遷移）
  • 勤怠詳細（修正申請フォーム）
  • 申請一覧（承認待ち / 承認済み）

管理者機能

  • 管理者ログイン / ログアウト（Fortify）
  • 全ユーザーの日次勤怠一覧（前日・翌日切り替え）
  • スタッフ別月次勤怠一覧（前月・翌月切り替え、CSV出力）
  • 勤怠詳細の確認 / 直接修正（バリデーション付き）
  • 修正申請一覧（承認待ち / 承認済み）
  • 修正申請詳細 / 承認処理
  • スタッフ一覧（氏名・メールアドレス表示）

使用技術

  • Laravel 9 / PHP 8.x
  • Docker / Docker Compose
  • MySQL
  • Bladeテンプレート + CSS / JavaScript
  • npm / Node.js（リアルタイム時計処理）
  • GitHub

今後の改善・追加予定

  • カレンダー機能（シフトや勤務日をカレンダー表示）
  • 未打刻に対応した新規レコード作成機能
  • 勤怠時間のグラフ化・統計表示
